# 💰 FinMate – 개인 맞춤형 금융 코치 서비스

FinMate는 사용자의 소비 습관, 금융 성향, 소득 등을 분석하여  
적합한 금융상품을 추천하고, 가계부 관리 및 만기 알림까지 제공하는  
**개인 맞춤형 금융 관리 서비스**입니다.

---

## 📌 1. 서비스 개요 (Overview)

현대인은 다양한 금융상품과 복잡한 소비 패턴 속에서  
**올바른 재무 의사결정**을 내리기 어렵습니다.  
FinMate는 사용자의 데이터를 기반으로 금융 생활 전반을 관리해주는  
**AI 기반 금융 코치** 역할을 목표로 합니다.

### 🎯 목표
- 소비 습관 분석 기반 맞춤형 금융상품 추천
- 가계부 자동화 및 월별 소비 리포트 제공
- 금융상품 만기 관리 및 대체 상품 제안
- 사용자 맞춤형 금융관리 경험 제공

## 🚀 3. 주요 기능 (Key Features)

### 👤 회원 기능
- 회원가입, 로그인, 로그아웃
- 프로필 입력 (소득, 목표, 리스크 성향)

### 💸 가계부 기능
- 수입/지출 입력 (카테고리 분류)
- 월/주/일 기준 내역 조회
- 소비 분석 그래프 제공

### 🏦 금융상품 기능
- 예금/적금 상품 검색 (은행, 금리, 기간 필터)
- 금융상품 상세 정보 제공
- 공공데이터 기반 API 연동

### 🤖 AI 기반 추천 기능
- 소비 패턴 기반 맞춤형 추천
- 적금/예금 만기 알림
- 만기 대체 상품 TOP3 추천

### 📊 대시보드
- 월 요약 정보 (수입/지출/잔액)
- 소비 그래프
- 추천 금융상품
- 만기 예정 알림

## 📱 4. 주요 페이지 구성 (Pages)

| 페이지 | 설명 |
|--------|-------|
| 로그인 / 회원가입 | 사용자 인증 화면 |
| 대시보드 | 월별 요약 + 추천 + 만기 정보 |

| 금융상품 상세 | 금리/조건/가입 정보 + 추천 사유 |
| 내 금융상품 관리 | 만기 일정, 보유 상품 조회 |
